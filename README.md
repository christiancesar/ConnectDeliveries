# ConnectDeliveries :triangular_flag_on_post:
  
  O intuito deste projeto é fornecer facilidade de integração das Apis de Deliverys para a linguagem Delphi. Foque em sua regra de negócio e deixe o restante conosco!

# Begin :beginner:
  
  Passo 1: Baixe o repositorio
  ```bash
  $ https://github.com/christiancesar/ConnectDeliveries.git 
  ```
  Passo 2: Aponte no seu LibraryPath as Units `\.src\`
  
  Passo 3: Pronto!

  ### **Para auxiliar em seu projeto e ajudar a entender nosso sdk, execute o projeto ConnectDeliveries.dproj**
